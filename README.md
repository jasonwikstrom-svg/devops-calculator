
# My DevOps-app

A simple web-based calculator built with Vite. The app is tested with Vitest and automatically deployed using Github Actions to Github Pages.

---

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Users can input their own numbers

---

## Technologies
- JavaScript
- Vite
- Vitest
- Github Action (CI/CD)
- Github Pages

---

## Installation
```bash
npm install
```

## Run locally
```bash
npm run dev
```

## Run tests
```bash
npm run test
```

## Build project
```bash
npm run build
```

---

## CI/CD
This project uses Github Actions to automatically:
- run tests on push to main
- build the project
- deploy to Github Pages
